Operator Manual

This manual is for running EtherealAI as a non-coder CEO/operator. Use Simple Operator Mode for normal work. Use Advanced Developer Mode only when intentionally reviewing raw logs, records, variables, or diagnostics.

Start Here Walkthrough

1. Open Mission Control Check Core Setup, Paper Trading, Wallet Metadata, Security, and Live Trading Lock status. Open Mission Control
2. Ask What Should I Do Next The assistant uses local state and will not route you to optional API/provider keys after paper trading is ready. Use Next Action
3. Run Local Paper Trading Paper trading does not require exchange APIs, live trading, wallet signing, seed phrases, private keys, or terminal commands. Open Paper Trading
4. Add Wallet Metadata Add public wallet addresses through the UI only. Never enter wallet passwords, seed phrases, private keys, or recovery phrases. Open Wallet & Funding

Operator Manual

Mission Control Use this first. It answers what is ready, what is locked, what needs review, and what button to press next.
Setup Wizard Required Now contains only true local paper-trading requirements. Optional Later contains APIs, socials, GitHub, Cloudflare, exchanges, and provider keys.
Strategy / Paper / Bots Follow the guided flow: create strategy, run backtest, create paper plan, start paper schedule, review results.
Wallet & Funding Public metadata only. Hardware wallet signing and live approvals are future locked-mode workflows.
Security Read Safe, Review Needed, Fix Now, and Optional Later owner tasks. Raw system internals are hidden in Simple Mode.
Solidity, Social Ops, Creator These create local plans, drafts, specs, and blueprints. They do not deploy tokens, post publicly, mutate DNS, or buy domains in Simple Mode.

Safety Rules

Never paste wallet secrets No seed phrase, recovery phrase, private key, wallet password, or passphrase belongs in EtherealAI.
Optional is not failed Missing exchange APIs, social tokens, GitHub, Cloudflare, Polygon RPC, and similar future integrations are not setup failures.
Live trading is locked Live orders, wallet signing, token deployment, external posting, and DNS mutation remain disabled until a future high-security approval process exists.
No terminal commands for normal use Operate through buttons, guided flows, and visible forms in Simple Operator Mode.
